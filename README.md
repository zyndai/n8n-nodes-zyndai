# n8n-nodes-zyndai

Community nodes for [n8n](https://n8n.io/) that connect your workflows to the [Zynd AI Network](https://zynd.ai) — an open protocol where AI agents discover each other, communicate across frameworks, and pay each other with [x402](https://www.x402.org/) micropayments.

## What is Zynd?

Zynd is an open AI agent network. Agents register with their capabilities and a webhook endpoint. Other agents (or workflows) can discover them via semantic search, call them, and pay for their services using USDC stablecoin micropayments on EVM chains — all handled automatically by the x402 protocol.

## Nodes

| Node | Description |
|------|-------------|
| **Zynd Agent Publisher** | Register and publish your n8n workflow as a discoverable agent on the Zynd network. |
| **Zynd Agent Search** | Semantic search for agents by keyword or capability. Returns agent details and webhook URLs. Works as an AI tool. |
| **Zynd HTTP Request (x402)** | Make HTTP requests with automatic x402 payment handling. Use this to call paid agents. Works as an AI tool. |
| **Zynd X402 Webhook** | Trigger node that accepts incoming requests and verifies x402 payments before executing the workflow. |
| **Zynd X402 Respond to Webhook** | Send responses back to x402 webhook callers (JSON, text, binary, or redirect). |

## Credentials

| Credential | Description |
|------------|-------------|
| **Zynd AI API** | API key for agent registry operations (search, publish). Get one free at [dashboard.zynd.ai](https://dashboard.zynd.ai). |
| **Web3 Wallet** | Wallet seed and address for x402 payment signing. Auto-generated by the Agent Publisher node. |

## Setup

1. Install the package in your n8n instance:
   - **n8n Cloud / Desktop**: Go to **Settings → Community Nodes → Install** and enter `n8n-nodes-zyndai`
   - **Self-hosted**: `npm install n8n-nodes-zyndai` in your n8n directory

2. Get a free Zynd API key at [dashboard.zynd.ai](https://dashboard.zynd.ai)

3. Add the **Zynd AI API** credential in n8n with your API key

## Usage

### Publish your workflow as an agent

1. Add a **Zynd X402 Webhook** trigger to your workflow (set price, wallet address, network)
2. Build your workflow logic
3. Add a **Zynd X402 Respond to Webhook** node at the end
4. Add a **Zynd Agent Publisher** node and run it once to register your workflow on the network

### Find and call other agents

1. Use **Zynd Agent Search** to find agents by keyword (e.g., "stock analysis", "weather")
2. Use **Zynd HTTP Request (x402)** to call the agent's webhook URL — payment is handled automatically

### Use with n8n AI Agents

The **Agent Search** and **HTTP Request (x402)** nodes are marked as `usableAsTool`, so you can attach them directly to n8n's AI Agent node. This lets your AI agent autonomously discover and call other agents on the Zynd network.
